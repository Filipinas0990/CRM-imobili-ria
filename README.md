# CRM Imobiliário

Sistema web de gestão para imobiliárias e corretores, focado em organização de leads, controle de visitas, follow-ups e vendas.

## 🚀 Visão geral
Este projeto foi criado para resolver um problema comum em imobiliárias: a perda de oportunidades por falta de controle e acompanhamento de leads.

### Principal objetivo
- Organizar o fluxo de leads
- Gerenciar visitas e atendimentos
- Controlar follow-ups e retornos
- Acompanhar negocociações e vendas

## 🧩 Tecnologias usadas
- React 18
- Vite
- TypeScript
- Tailwind CSS
- Supabase
- React Query
- Radix UI
- Zustand
- Framer Motion

## 📁 Estrutura do projeto
- `src/` — código-fonte da aplicação
- `src/pages/` — páginas principais do app
- `src/components/` — componentes reutilizáveis e UI
- `src/lib/` — integrações e utilitários
- `src/services/` — lógica de chamadas à API e serviços
- `supabase/` — configuração e funções do Supabase

## ▶️ Como executar localmente
```bash
npm install
npm run dev
```

Acesse `http://localhost:5173` após iniciar o servidor.

## 📦 Scripts disponíveis
- `npm run dev` — inicia o servidor de desenvolvimento
- `npm run build` — cria a build de produção
- `npm run build:dev` — cria a build em modo development
- `npm run preview` — pré-visualiza a build de produção
- `npm run lint` — executa o ESLint no projeto

## 🔧 Configuração
Verifique as variáveis de ambiente e a integração com o Supabase na pasta `supabase/` e em `src/lib/supabaseClient.js`.

## 🎯 Funcionalidades principais
- Gestão de leads
- Pipeline de vendas
- Registro de visitas
- Follow-ups e agenda
- Relatórios e dashboards
- Autenticação e controle de acesso
